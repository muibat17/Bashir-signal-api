# Bashir Crypto Signal API ✅

This is a fully–automated crypto market scanner that tracks **50 high-volume Binance USDT pairs** across multiple timeframes (1m, 5m, 15m).

It generates:
- ✅ Buy signals (LONG)
- ✅ Sell signals (SHORT)
- ✅ Entry price
- ✅ Stop-loss (SL)
- ✅ Take profit 1 (TP1)
- ✅ Take profit 2 (TP2)
- ✅ Quality score
- ✅ AI analysis (local or GPT)
